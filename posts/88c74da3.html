Lecture 01b — Introduction

核心问题

这门课在研究什么？为什么要用网络/图来描述现实世界？

核心定义

网络（network）可以看作：

其中： - 是节点集合 - 是边集合

节点代表对象，边代表对象之间的关系。

为什么网络重要

  • 网络是描述复杂系统的通用语言。
  • 很多不同领域的数据都能转成图：
    • 社交网络
    • 航空网络
    • Web 图
    • 引文网络
    • 生物网络
  • 不同领域虽然语义不同，但图结构问题往往相似。

课程关注点

课程会关注大规模网络中的： - 图结构基础 - tie strength - structural balance - network evolution - community detection - hubs/authorities - PageRank

图论基础回顾

这节通常也会作为图论预热，默认你熟悉： - degree - path - distance - connected component - directed/undirected graph - weighted/unweighted graph

课程运行信息

  • 讲师、tutorial、Canvas、Ed 讨论区等信息都在这节说明。
  • 评估、作业、通知主要通过 Canvas 发布。

takeaway

这节的目的不是公式推导，而是建立课程地图： > 大规模网络分析 = 用图论语言研究真实复杂系统的结构、演化和算法。
