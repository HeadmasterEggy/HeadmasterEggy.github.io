🚀 每日科技速递 - 2026-03-21

过去 48 小时里，最值得盯住的主线仍是 OpenAI 收购 Astral：这不是单纯买一个热门开源团队，而是在把 Python 工具链直接并入 Codex 的执行层，进一步把“写代码”推进到“跑完整开发流程”。与此同时，研究圈和产业侧也在同步升温：一边是 ICLR / DeepMind 相关讨论继续暴露 AI 人才与评审体系的高压现实，另一边则是 Supermicro GPU 走私指控把 AI 基础设施与出口管制再次拉回聚光灯。中文模型侧也有新变量，Qwen3.5-Max 预览版在 LMArena 冲上中国最强梯队，说明国内旗舰模型竞争还在继续加速。

🧠 LLM / Large Models

• 🔥15 | OpenAI 收购 Astral，Codex 开始吃进 Python 基础设施层：官方公告把目标说得很直白——不只让 AI 生成代码，而是进一步参与依赖管理、格式化、类型检查、运行验证和长期维护。Astral 的 uv、Ruff、ty 已经是大量 Python 工作流的底座，这笔交易等于把开发工具链直接接到模型执行回路里。OpenAI Blog
